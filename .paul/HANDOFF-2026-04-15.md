## Critical Patterns (must follow exactly)

- `ColumnDef` imported from `@shared/ui/data-table` — NOT from barrel
- `const { toast } = useToast()` — not standalone import
- `useEffect` reset on `[open, entity]` — both create (null) and edit (non-null) cases
- `mutateAsync` in try/catch — close + toast only on success; dialog stays open on error
- `onOpenChange` ignores close when `isPending` (dialog close guard)
- Inline list error state: `isError` → error message + retry button (not empty table)
- `mapRoleError`: access `(err as Record<string, unknown>)?.details` not `err.message` for constraint check
