<objective>
## Goal
Standardize all action dropdown items across every list page: unified Vietnamese labels, Lucide icons on every item, and a separator before the destructive Delete action.

## Purpose
Currently the 9 DropdownMenu-based pages use inconsistent edit labels ("Sửa" vs "Chỉnh sửa"), have no icons on any items, and no separator before the red Xóa item. The 2 inline-button pages (bookings, payments) show a plain "Xem" button with no icon. This creates a fragmented feel when navigating between modules.

## Output
- All 9 dropdown pages: consistent "Chỉnh sửa" label + Pencil icon; Trash2 icon on Xóa; DropdownMenuSeparator before Xóa
- Trips: Pencil icon added to "Chỉnh sửa"; "Phân công" + Users icon already correct — keep as is
- Routes: MapPin icon added to "Điểm dừng"
- Bookings + Payments: Eye icon added to inline "Xem" button
</objective>
