## DB Schema (employees table)
- `id` uuid PK gen_random_uuid()
- `user_id` uuid UNIQUE nullable → profiles.id ON DELETE SET NULL
  (constraint name: `employees_user_id_key`)
- `hire_date` date nullable
- `license_number` text UNIQUE nullable
  (constraint name: `employees_license_number_key`)
- `license_expiry` date nullable — used for 30-day expiry alert
- `is_active` boolean NOT NULL default true
- `created_at`, `updated_at` timestamptz auto-managed

## DB Schema (user_roles table — role assignment)
- `user_id` uuid FK → profiles.id ON DELETE CASCADE (PK part 1)
- `role_id` uuid FK → roles.id ON DELETE CASCADE (PK part 2)
- `created_at` timestamptz

## DB Schema (profiles table — read-only, created by auth trigger)
- `id` uuid PK (= auth.users.id)
- `full_name` text nullable
- `email` text NOT NULL
- `phone` text nullable
- `avatar_url` text nullable

## DB Relationships affecting this plan
- `employees.user_id` → `profiles.id`: PostgREST nested select `profiles(id, full_name, email, phone)`
- `trip_staff.employee_id` → `employees.id` ON DELETE RESTRICT → 23503 on employee delete
- `user_roles` links profiles → roles (not directly linked to employees; bridge via employees.user_id = user_roles.user_id)

    **employee.api.ts** — Supabase CRUD:
    - `fetchEmployees(params)`:
      - When search present: use `profiles!inner(id, full_name, email, phone)` and `.ilike('profiles.full_name', '%q%')`
      - When no search: use `profiles(id, full_name, email, phone)` (LEFT JOIN — shows employees without profiles too)
      - Apply `isActive` filter: `.eq('is_active', isActive)` when defined
      - `.order('created_at', { ascending: false })`, `.range(from, to)`, count: exact
      - Returns `{ data: Employee[], count: number }`

    - `fetchEmployee(id)`:
      ```typescript
      supabase.from('employees')
        .select('*, profiles(id, full_name, email, phone)')
        .eq('id', id).single()
      ```

    - `fetchProfiles()` — for user dropdown in form (no pagination needed for MVP, fetch up to 1000):
      ```typescript
      // Returns all profiles sorted by full_name for the user dropdown
      const { data, error } = await supabase
        .from('profiles')
        .select('id, full_name, email')
        .order('full_name', { ascending: true })
        .limit(1000);
      ```
      Returns `EmployeeProfile[]`

    - `fetchEmployeeRole(userId: string)` — get current role assignment for a user:
      ```typescript
      const { data } = await supabase
        .from('user_roles')
        .select('role_id')
        .eq('user_id', userId)
        .maybeSingle();
      return data?.role_id ?? null;
      ```

    - `createEmployee(input: EmployeeInsert)` — insert + select + single; throw on error

    - `updateEmployee(id: string, input: EmployeeUpdate)` — update + select + single; throw on error

    - `deleteEmployee(id: string)` — delete by id; throw on error

    - `assignEmployeeRole(userId: string, roleId: string | null)`:
      ```typescript
      // Delete existing role assignments for user first
      await supabase.from('user_roles').delete().eq('user_id', userId);
      // Then insert new one if roleId provided
      if (roleId) {
        const { error } = await supabase
          .from('user_roles')
          .insert({ user_id: userId, role_id: roleId });
        if (error) throw error;
      }
      ```

    **employee-form-schema.ts** — Zod schema + error mapper:
    ```typescript
    export const employeeFormSchema = z.object({
      user_id: z.string().uuid('Vui lòng chọn người dùng').or(z.literal('').transform(() => null)).nullable(),
      hire_date: z.string().regex(/^\d{4}-\d{2}-\d{2}$/, 'Ngày không hợp lệ').nullable().or(z.literal('').transform(() => null)),
      license_number: z.string().max(50).nullable().or(z.literal('').transform(() => null)),
      license_expiry: z.string().regex(/^\d{4}-\d{2}-\d{2}$/, 'Ngày không hợp lệ').nullable().or(z.literal('').transform(() => null)),
      is_active: z.boolean().default(true),
      role_id: z.string().uuid().nullable().or(z.literal('').transform(() => null)),
    });
    export type EmployeeFormValues = z.infer<typeof employeeFormSchema>;

    export function mapEmployeeError(err: unknown): string {
      const msg = err instanceof Error ? err.message : String(err);
      if (msg.includes('employees_license_number_key') || (msg.includes('23505') && msg.includes('license_number')))
        return 'Số bằng lái đã tồn tại trong hệ thống';
      if (msg.includes('employees_user_id_key') || (msg.includes('23505') && msg.includes('user_id')))
        return 'Người dùng này đã có hồ sơ nhân viên';
      if (msg.includes('23503') && msg.includes('trip_staff'))
        return 'Nhân viên đã được phân công chuyến đi, không thể xóa';
      return 'Đã xảy ra lỗi, vui lòng thử lại';
    }
    ```

      function getLicenseExpiryBadge(expiryDate: string | null) {
        if (!expiryDate) return null;
        const today = dayjs().startOf('day');
        const expiry = dayjs(expiryDate);
        const daysLeft = expiry.diff(today, 'day');
        if (daysLeft < 0) return <Badge variant="destructive">Hết hạn</Badge>;
        if (daysLeft <= 30) return <Badge variant="warning">Sắp hết hạn</Badge>;
        return null;
