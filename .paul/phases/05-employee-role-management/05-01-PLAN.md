## DB Schema (roles table)
- `id` uuid PK gen_random_uuid()
- `name` text NOT NULL UNIQUE (constraint name: `roles_name_key`)
- `description` text nullable
- `permissions` jsonb NOT NULL default '[]' CHECK(jsonb_typeof = 'array')
- `created_at`, `updated_at` timestamptz auto-managed

## DB Relationships
- `user_roles.role_id` FK → `roles.id` ON DELETE CASCADE
  → Deleting a role cascades to user_roles (no 23503 on delete; warn user in dialog instead)

## Default roles in DB (seed data — read-only, should not be deleted carelessly)
admin, fleet_manager, dispatcher, ticketing_agent, driver, assistant

    **role-form-schema.ts** — Zod schema + error mapper:
    ```typescript
    export const roleFormSchema = z.object({
      name: z.string().min(1, 'Tên vai trò là bắt buộc').max(100),
      description: z.string().max(500).optional().or(z.literal('')),
      // audit-added SR-2: validate each permission string — no empty/whitespace values reach DB
      permissions: z.array(
        z.string()
          .min(1, 'Quyền không được để trống')
          .regex(/^\S+$/, 'Quyền không được chứa khoảng trắng')
      ),
    });
    export type RoleFormValues = z.infer<typeof roleFormSchema>;

    // audit-added MH-1: use .details field (not raw message) for 23505 disambiguation
    // audit-added SR-3: auth-expiry PGRST301/401/403 path
    // audit-added SR-5: 23514 CHECK constraint on permissions JSONB
    export function mapRoleError(err: unknown): string {
      const e = err as Record<string, unknown>;
      const msg = (e?.message as string) ?? String(err);
      const details = e?.details as string | undefined;
      const code = e?.code as string | undefined;
      if (code === 'PGRST301' || code === '401' || code === '403')
        return 'Phiên đăng nhập hết hạn, vui lòng đăng nhập lại';
      if (msg.includes('roles_name_key') || details?.includes('(name)'))
        return 'Tên vai trò đã tồn tại, vui lòng chọn tên khác';
      if (code === '23514' || msg.includes('23514'))
        return 'Dữ liệu quyền không hợp lệ, vui lòng thử lại';
      return 'Đã xảy ra lỗi, vui lòng thử lại';
    }
    ```
