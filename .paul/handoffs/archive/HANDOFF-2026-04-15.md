- Types: `Station`, `StationInsert`, `StationUpdate`, `StationListParams`
- Hooks: `useStations`, `useStation`, `useCreateStation`, `useUpdateStation`, `useDeleteStation`

**Routes DB schema (migration 20260411140000):**
- `name text NOT NULL UNIQUE` — constraint: `routes_name_key`
- `origin_station_id uuid NOT NULL REFERENCES stations(id) RESTRICT` — constraint: `routes_origin_station_fk`
- `destination_station_id uuid NOT NULL REFERENCES stations(id) RESTRICT` — constraint: `routes_destination_station_fk`
- `distance_km numeric(8,2) NOT NULL` — CHECK: > 0
- `estimated_duration interval NOT NULL` — CHECK: > '0'::interval
- `base_price numeric(12,2) NOT NULL DEFAULT 0` — CHECK: >= 0
- `is_active boolean NOT NULL DEFAULT true`
- CHECK: `routes_different_origin_destination` (origin ≠ destination)
- DELETE RESTRICT from trips (trips.route_id FK)

**PostgREST FK join syntax (verified):**
```
'*, origin_station:stations!routes_origin_station_fk(id,name), destination_station:stations!routes_destination_station_fk(id,name)'
```
