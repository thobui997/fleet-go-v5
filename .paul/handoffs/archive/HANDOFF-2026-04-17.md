## Design Implementation Notes

### ModernCalendar Component
- Custom implementation replacing react-day-picker's default Calendar
- Vietnamese locale support with `vi` from date-fns
- Weekday labels: ['CN', 'T2', 'T3', 'T4', 'T5', 'T6', 'T7']
- 42-day grid (6 rows × 7 days) for consistent layout
- Click month/year to jump to today
- ChevronLeft/ChevronRight navigation with strokeWidth={2}

### DatePicker Button/Trigger
- `justify-between` (icon on right)
- `border-border/50` for subtle border
- `hover:bg-accent/50` for subtle hover
- Selected value: `font-medium`
- Placeholder: `text-muted-foreground/50 font-normal`
- CalendarIcon: `opacity-50`

### DateTimePicker Time Input
- Clock icon: positioned left with `absolute left-3`
- Width: `w-[132px]`
- Same border/hover/focus styling as date trigger
- Helper text shows when incomplete (date only OR time only)
